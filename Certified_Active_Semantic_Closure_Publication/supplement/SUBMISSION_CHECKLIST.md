# Anonymous submission checklist

## Scientific content

- [x] The target problem is stated independently of repository terminology.
- [x] The central no-go and positive construction are both formalized.
- [x] The missing aliasing-to-sufficiency bridge is a generic theorem.
- [x] Detection, informative query, strict reduction, intrinsic successor,
      closure, and preservation are proved on the published trajectories.
- [x] Finite termination and open progress are distinguished.
- [x] Neural catalogue certification is separated from generalization.
- [x] Limitations and non-claims are explicit.
- [x] Related work is represented by primary sources.
- [x] Structural related-work comparison and priority boundary are explicit.
- [x] Adaptive public no-go, global characterization, composable synthesis,
      exact posterior, and four necessity countermodels are formalized.
- [x] A complete positive instance establishes that the generic synthesis
      interfaces are inhabited.
- [x] Exact adapters connect the three finite repairs and every open stage to
      the public binary protocol without enlarging their quantifiers.

## Formal artifact

- [x] Standalone Lake project and pinned Lean toolchain are present.
- [x] The publication entry point imports no parent-project module.
- [x] Exactly one final axiom-audit block is present in every new Lean file.
- [x] All audit names are concrete declarations.
- [x] No `sorry`, `admit`, introduced axiom, `Classical`, `propext`, or
      `Quot.sound` is used in the publication module.
- [x] The final aggregate exposes all headline guarantees.
- [x] A clean-build command and staged verification commands are documented.
- [x] The checkpoint SHA-256 and evidential status are documented.
- [x] Clean publication and whole-library builds are recorded as successful.

## Manuscript package

- [x] English manuscript is complete.
- [x] French scientific summary is included.
- [x] Technical appendix is complete.
- [x] Claim–evidence matrix is normative.
- [x] Source-problem resolution audit and exact non-claims are explicit.
- [x] Reproducibility protocol is included.
- [x] Bibliography is machine-readable BibTeX.
- [x] Anonymous citation metadata is present.
- [x] Version and review license notice are present.

## Post-acceptance metadata only

The following substitutions do not change scientific content and are excluded
from the blind package: author names, affiliations, acknowledgements, venue
template, repository release URL, archival DOI, and the copyright holders’
chosen public distribution license.
